# TypeScript Plugin

The typescript plugin provides versionator support for TypeScript projects.

## Overview

TypeScript projects benefit from typed version exports. Like JavaScript, TypeScript projects use `package.json` for version management:

```json
{
  "name": "mypackage",
  "version": "1.2.3"
}
```

## Injection Methods

| Method | Description |
|--------|-------------|
| `emit` | Generate `version.ts` with typed version exports |
| `patch` | Update `version` field in `package.json` |

### Emit

Generate a TypeScript module with typed exports:

```bash
versionator emit typescript
# Creates version.ts
```

Generated file:

```typescript
// Auto-generated by versionator. Do not edit.

export const VERSION: string = "1.2.3";
export const MAJOR: number = 1;
export const MINOR: number = 2;
export const PATCH: number = 3;
export const GIT_HASH: string = "abc1234";
```

Usage:

```typescript
import { VERSION } from './version';

console.log(`Version: ${VERSION}`);
```

### Patch

Update the version directly in `package.json`:

```bash
versionator patch typescript
```

## Project Integration

### Add to .gitignore

If generating at build time:

```gitignore
# Generated version file
version.ts
```

### Export from Package

In your main entry point:

```typescript
// index.ts
export { VERSION } from './version';
```

## Build System Integration

### npm scripts

```json
{
  "scripts": {
    "prebuild": "versionator emit typescript",
    "build": "tsc",
    "version:bump": "versionator patch increment && versionator patch typescript"
  }
}
```

### CI/CD (GitHub Actions)

```yaml
- name: Generate version
  run: versionator emit typescript

- name: Build
  run: npm run build

- name: Publish
  run: npm publish
```

## Configuration

Default output path: `version.ts`

Override with:

```bash
versionator emit typescript --output src/version.ts
```

## ES Modules vs CommonJS

The generated file uses ES module syntax (`export`). For CommonJS projects, use the [JavaScript plugin](../javascript/) or configure TypeScript to compile to CommonJS.
