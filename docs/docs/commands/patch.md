# patch

Manage patch version

Commands to increment or decrement the patch version component

## Usage

```bash
versionator patch [command]
```

## Subcommands

| Command | Description |
|---------|-------------|
| `decrement` | Decrement patch version |
| `increment` | Increment patch version |

### decrement

Decrement patch version

Decrement the patch version

```bash
versionator patch decrement
```

**Aliases:** `dec`, `-`

### increment

Increment patch version

Increment the patch version

```bash
versionator patch increment
```

**Aliases:** `inc`, `+`
